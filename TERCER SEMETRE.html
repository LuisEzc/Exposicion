¿Qué es el software de aplicación?
Es un tipo de software que permite al usuario realizar tareas específicas como redactar textos, hacer cálculos, editar imágenes, o gestionar información. Está diseñado para interactuar directamente con el usuario, a diferencia del software del sistema que gestiona el hardware.
Características del software de aplicación

Interfaz amigable para el usuario.

Orientado a tareas concretas, como escribir, calcular, diseñar, etc.

Requiere un sistema operativo para funcionar (como Windows, macOS o Linux).

Ejemplos comunes: Microsoft Word, Excel, PowerPoint, Google Docs, Adobe Photoshop, etc.
Gestión de archivos de texto y hojas de cálculo
1. Microsoft Word (Procesador de texto)

Función principal: Crear y editar documentos de texto.

Usos comunes: Informes, cartas, ensayos, currículos, folletos.

Funciones destacadas:

Formato de texto (negrita, cursiva, subrayado).

Inserción de imágenes, tablas, ecuaciones, encabezados y pies de página.

Control de cambios y comentarios para trabajo colaborativo.

Convertir a PDF fácilmente.

Plantillas prediseñadas.

Extensiones comunes: .docx, .doc, .pdf, .rtf.
2. Microsoft Excel (Hoja de cálculo)

Función principal: Manipulación de datos numéricos y textuales en celdas.

Usos comunes: Presupuestos, contabilidad, estadísticas, gráficos.

Funciones destacadas:

Fórmulas y funciones (SUMA, PROMEDIO, BUSCARV, SI, etc.).

Gráficos dinámicos (barras, líneas, tortas).

Tablas dinámicas para analizar grandes volúmenes de datos.

Validación de datos y filtros automáticos.
Automatización con macros (VBA).

Extensiones comunes: .xlsx, .xls, .csv.
Ventajas del uso de Word y Excel

Aumentan la productividad.

Facilitan la organización de la información.

Permiten compartir y colaborar en documentos fácilmente.

Son estándares en el entorno profesional y académico.
