Submódulo 1:
Sistemas de Información
En este submódulo se trabajó el concepto de sistemas de información y su aplicación en diferentes contextos. Comenzamos analizando problemas reales (como la gestión de alumnos o productos) para diseñar un sistema que los resolviera utilizando
bases de datos y formularios.
Aprendimos sobre el ciclo de vida del software: análisis, diseño, desarrollo, prueba, implementación y mantenimiento. Utilizamos bases de datos relacionales donde creamos tablas, relaciones entre datos, consultas, formularios e informes. Esta base es indispensable para estructurar la información de manera ordenada y lógica.
Herramientas utilizadas:
• Microsoft Access
• LibreOffice Base
• Diagramas de flujo (draw io, Lucidchart)
·Diagramas de tijo (draw.io, Lucidchart)
Actividades realizadas:
• Diseño de base de datos para control escolar.
• Creación de formularios e informes interactivos.
• Propuesta de solución para problema real mediante sistema.
Aprendizajes adquiridos:
• Aplicar una metodología estructurada para el desarrollo de sistemas.
• Organizar información en tablas relacionadas.
• Generar soluciones informáticas funcionales en diferentes entornos
Submódulo 2: Programación
Aqui desarrollamos habilidades para programar de manera lógica y estructurada. Empezamos con algoritmos, diagramas de flujo y pseudocódigo para entender la lógica detrás de un programa, y luego lo llevamos a la práctica con lenguajes como C+*, Python
o pseudolenguajes visuales.
Aprendimos a declarar variables, usar operadores, estructuras de decisión (if, else), ciclos (while, for) y arreglos. Este submódulo fomenta la resolución de problemas y la creatividad, asi como la perseverancia ante errores de código.
Herramientas utilizadas:
• Dev C++ / Code:: Blocks
• Visual Basic
• PSelnt
Actividades realizadas:
• Programas con decisiones y ciclos (calculadoras, menús, validaciones).
• Algoritmos para resolver problemas matemáticos o escolares.
Aprendizajes adquiridos:
  • Pensamiento lógico y resolución de problemas.
• Sintaxis básica de un lenguaje de programación.
• Capacidad de convertir una necesidad en una solución computacional.
• Submódulo 3: Programación Visual
En este submódulo llevamos la programación a un entorno visual, donde los programas no solo tienen lógica, sino también interfaz gráfica. Trabajamos con herramientas como Visual Basic o entornos similares para crear ventanas, botones, formularios y
acciones interactivas.
Aprendimos a manipular eventos como "clic", "pasar el mouse", "cerrar ventana"
entre otros. Tambien combinamos los
conocimientos de bases de datos y lógica para desarrollar aplicaciones completas
Herramientas utilizadas:
• Visual Studio (VB.NET o C#)
= App Inventor (basico)
Gambas o Scratch (alternativas didácticas)
Actividades realizadas:
• Mini sistema de registro con formularios
• Aplicación con botones, etiquetas y funciones personalizadas
  • Proyecto final: aplicación completa (calculadora, agenda, encuesta).

Aprendizajes adquiridos:
• Crear aplicaciones con interfaz de usuario.
Comprender el ciclo de eventos en una app visual.
• Unir programación y diseño para soluciones prácticas.
